Scores
==================================

Description
-----------

The Scores are used to specify the score values corresponding to somethings you want to visualized.
If you want just to see the API specification, please skip to
[Classes section](tf_keras_vis.utils.scores.html#module-tf_keras_vis.utils.scores).

-----------

To use [Feature visualization]() or [Attentions](), we need the score instance or a function you
defined to calculate the score value.

* Binary classification model
