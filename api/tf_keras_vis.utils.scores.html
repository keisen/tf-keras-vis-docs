Scores¶

Description¶

The Scores are used to specify the score values corresponding to somethings you want to visualized. If you want just to see the API specification, please skip to Classes section.

How to use¶

To use Feature visualization or Attentions, we need the score instance or a function you defined to calculate the score value.

  • Categorical classification model

  • Binary classification model

  • Regression model

class tf_keras_vis.utils.scores.InactiveScore[source]¶

Bases: tf_keras_vis.utils.scores.Score

A score function that deactivate model output passed to __call__().

With a multiple output model, you can use this if you want a output to be excluded from targets of calculating gradients.
